## PDF to Audio Converter using Python

## Description
This project is a Python application that reads PDF files and converts their text content into audio.
It allows users to select a PDF file and listen to its contents using text-to-speech technology.

## Features
- Select PDF file using file dialog
- Extracts text from PDF pages
- Converts PDF text into speech
- Beginner-friendly project

## Technologies Used
- Python
- PyPDF2
- pyttsx3
- Tkinter

## How to Run
1. Install required libraries:
pip install PyPDF2 pyttsx3
